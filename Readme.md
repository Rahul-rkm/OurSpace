# OurSpace - a social media platform.

[![Our-Space-Homepage.jpg](https://i.postimg.cc/MHsHQkBL/Our-Space-Homepage.jpg)](https://postimg.cc/RJH4kDX1)

## Features

- Create your account with email.
- Add profile picture.
- Post pictures and text content.
- Follow other people.
- Like other's posts.

## Tech Stack

- MongoDB
- express
- React JS
- NodeJS
