# OurSpace - a social media platform.

![Image](./about/OurSpaceHomepage.jpg)

## Features

- Create your account with email.
- Add profile picture.
- Post pictures and text content.
- Follow other people.
- Like other's posts.

## Tech Stack

- MongoDB
- express
- React JS
- NodeJS
